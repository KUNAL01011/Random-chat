### Clone the project

```shell
git clone https://github.com/KUNAL01011/Random-chat.git
```

### Setup .env file

```js
PORT = ...
MONGO_DB_URI = ...
CLOUDINARY_CLOUD_NAME= ...
CLOUDINARY_API_KEY= ...
CLOUDINARY_API_SECRET= ...
JWT_SECRET = ...
NODE_ENV = development
```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm run dev
```
